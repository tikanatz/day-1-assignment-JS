Do's and Don'ts in JavaScript

Do's of JavaScript

  1. Can implement the same functionalities as HTML & CSS
  2. Can interact with AJAX, DOM, JSON, HTML, CSS
  3. Provide functionalities to user inputs

Don'ts of JavaScript

  1. Doesn't read or write on the Client's file system
  2. Cannot hide JavaScript
  3. Cannot close browser windows it did not open
